HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading elements should be used in order from h1 to h6, with h1 being used only once.
  • It is important to use features like alt to improve accesability and search engine hierarchy.
  • Elements can be distinguished by using the id or class tags.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Classes allow for more specified CSS styling.
  • Best practice is to link an external stylesheet to your HTML using a link tag in the head section
  • CSS can be used for positioning, color, style, and much more!

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the most up to date main branch
  • git add -A: stages all files in the working branch
  • git commit -m "add a short concise message here": commits the working directory
  • Confirm you are up to date by calling: git pull origin main
  • Push changes to the remote repository by calling: git push origin feature/feature-name

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Contril flow goes from top to bottom of the document.
  • An array can be used to store multiple types of data.
  • Arrays are zero-indexed, meaning the first element is number 0.
  • For loops can be used to iterate through all elements in an array.
  • Conditional statements add logic and perameters to your code.
  • Functions can be used to repeat code and save it to a variable.
  • Add JavaScript to your HTML using the script tag before the closing body tag.
